# Personal Website Visual Design Language

**Audience:** Product Leader & Technologist
**Design Principles:** Minimal · Modern · Clean · Analytical · Communicative · Creative

---

# 1. Brand Foundation

## Personality Traits

| Trait         | Visual Expression                               |
| ------------- | ----------------------------------------------- |
| Creative      | Subtle accent color, thoughtful visual elements |
| Analytical    | Clear grid layout, structured hierarchy         |
| Professional  | Neutral palette, restrained design              |
| Communicative | Large readable typography, generous whitespace  |

### Design Keywords

* Minimal
* Precise
* Human
* Strategic
* Modern

---

# 2. Color System

## Core Palette

| Role           | Color       | Hex     |
| -------------- | ----------- | ------- |
| Primary        | Deep Indigo | #1E3A8A |
| Secondary      | Teal        | #0EA5A4 |
| Accent         | Amber       | #F59E0B |
| Background     | Soft White  | #FAFAFA |
| Surface        | Light Gray  | #F3F4F6 |
| Text Primary   | Charcoal    | #111827 |
| Text Secondary | Slate       | #6B7280 |

## Color Usage

### Primary

Used for:

* Navigation
* Buttons
* Links
* Key UI elements

### Secondary

Used for:

* Data highlights
* Tags
* Supporting UI elements

### Accent

Used sparingly for:

* Calls to action
* Important metrics
* Interaction highlights

### Neutral Colors

Provide visual balance and support readability.

---

# 3. Typography

## Primary Typeface

**Lato**

Chosen for:

* Excellent readability
* Modern technology aesthetic
* Strong numeric rendering for metrics

## Secondary Typeface (Optional)

**Playfair Display**

Used for:

* Hero headlines
* Section titles
* Visual emphasis

---

# 4. Iconography

Icon style:

* Simple line icons
* Consistent stroke width
* Minimal visual weight

Guidelines:

* Stroke width: ~1.5px
* Monochrome
* Primary color used sparingly

---

# 5. Imagery Style

Visual tone should feel:

* thoughtful
* intelligent
* human

---

# 6. Motion & Interaction

Animations should feel **purposeful and subtle**.

---

# 7. Desired Experience

The website should feel like:

* A thoughtful product strategy document
* A modern technology interface
* A clean professional portfolio

Tone:

* Insightful
* Strategic
* Human
* Clear
* Confident
